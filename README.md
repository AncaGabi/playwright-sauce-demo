# 🎭 Playwright + TypeScript Automation Framework

This project contains automated end-to-end tests for the Sauce Demo application  
https://www.saucedemo.com/

The goal of this repository is to practice and deepen my knowledge of:

- Playwright
- TypeScript
- Page Object Model (POM)
- Test structure and framework design
- Environment configuration using `.env`
- Reusable components (Header / Footer)
- Test steps and reporting

---

## 📦 Tech Stack

- Playwright
- TypeScript
- Node.js
- dotenv (for environment variables)
